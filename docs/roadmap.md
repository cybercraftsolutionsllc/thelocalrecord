# Roadmap

## Phase 1

- Multi-tenant repo scaffold
- Public web app
- Registry-driven locality routing
- Implemented source adapters for key Manheim Township sources
- Basic persistence, diffing, classification, and review gating

## Phase 2

- Calendar and planning/zoning adapters
- Safer PDF extraction and confidence scoring
- Reviewer actions for approve, reject, and correction logging
- Durable artifact storage
- Additional municipalities
- Cloudflare-targeted production deployment

## Phase 3

- Better admin workflows
- Search and filtering
- Source health monitoring
- Expanded testing around adapter resilience
